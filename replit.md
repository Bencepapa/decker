# Decker Game

## Overview
Decker is a classic cyberpunk hacking game originally created in 2001 by Shawn Overcash, remade in JavaScript in 2018 by Palparepa. The game runs entirely in the browser as a static web application.

## Project Structure
- `index.html` - Main entry point
- `all.js` - Game logic and JavaScript code
- `all.css` - Styling
- `img/` - Game images and sprites
- `imgHelp/` - Help documentation images
- `snd/` - Sound effects (OGG format)
- `help.html` - Help documentation page

## Running the Project
The project is served as static files using `serve` on port 5000:
```
npx serve -l 5000 -s .
```

## Deployment
Configured for static deployment - all files in the root directory are served directly.
